How to connect to Braindrop

  1. Run the Braindrop server on a computer. (It is currently running, or else you wouldn't be reading this)
  2. Braindrop will show an address in the console after it runs
  3. Ensure the device you are wanting to use is connected to the same network as the device running the server
  4. Enter the address you receive from the Braindrop console into any device's web browser

You are now connected!
